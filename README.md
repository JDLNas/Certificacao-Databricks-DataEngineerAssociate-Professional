## 🐛 Solução de Problemas

### **Erro: "NameError: name 'x' is not defined"**
* Execute as células anteriores primeiro
* Variáveis são definidas em células anteriores

### **Erro: "SyntaxError: invalid syntax"**
* Verifique parênteses, aspas e indentação
* Python é sensível à indentação (use 4 espaços)

### **Célula não executa**
* Certifique-se de que o cluster está ativo
* Reinicie o kernel se necessário
